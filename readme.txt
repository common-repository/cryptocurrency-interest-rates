== Description ==
Free Cryptocurrency Interest Rates Widget from [EarnCryptoInterest.com](https://earncryptointerest.com). Use it to generate a **List of Top Cryptocurrency Interest Rates**. The widget can be customized with several parameters (number of displayed coins, dark/light theme etc.).

We are the only site which compiles cryptocurrency interest rates from different sources and makes them comparable. Currently we compile interest rates from 9 different sources. Rates are updated every 15 minutes.

== Installation ==
#### Install From WordPress Admin Panel

Please click in the Wordpress admin panel on **Plugins** and then on **Add New**. Search for **"Cryptocurrency Interest Rates"** and after finding our plugin, click on **Install**. Activate the plugin after installation.

#### Add crypto interest rates to your site

After installing and activating the plugin, please go to **Appearance/Widgets**. You will find there the widget **Cryptocurrency Interest Rates**. Just drag it to the right side to the desired position. You don't need to change any of the available parameters if you like how the widget is displayed on your site.
